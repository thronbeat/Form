1. What is the DOM?

The Document Object Model (DOM) is a programming interface that represents HTML documents as a tree of objects. JavaScript can manipulate these objects to change content, structure, and styling dynamically.

2. Selecting Elements

// Methods to select elements: document.getElementById('id') // Select by ID document.getElementsByClassName('class') // Select by class name document.getElementsByTagName('tag') // Select by tag name document.querySelector('selector') // Select first match (CSS selector) document.querySelectorAll('selector') // Select all matches
